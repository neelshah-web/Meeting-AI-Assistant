Expected Functionality

  • ✅ Floating button should appear in the top-right corner
  • ✅ Button should be draggable
  • ✅ Hover should show tooltip with platform info
  • ✅ Click should open recording interface
  • ✅ Button should persist when navigating pages

Troubleshooting

If the floating button is not visible:

  1. Reload this page (Ctrl+R or Cmd+R)
  2. Check if the extension is enabled in chrome://extensions/
  3. Look for any error messages in the browser console (F12)
  4. Try clicking the extension icon in the browser toolbar
